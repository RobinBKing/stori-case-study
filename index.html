Concept

A modern responsive web application for writers that simple, flexible and easy to use.

What if . . .

. . . a writer could go to a website that was simple to use, flexible in structure, but still provided all the tools needed for creating a killer novel.

Premise

I'm a web developer.
But I am also a writer.
I wrote my first novel without the help of writing software.

Back Story

My first novel took over 5 years! I had to created multiploe folders to organize my files and I often lost track of where things were, I frequently switched scenes around which involved a lot of cutting and pasting, and the word processor I used was basically overkill for what I needed.

Plot: The Challenge

I determined that for my next novel, I was going to get help. I have since bought and used existing software for writing, but they all came up short or cost more than I was willing to spend. Then I thought, hey I'm a writer, but I'm also a web developer. I'll just create what I need.

Goal: A responsive website that is simple, flexible and functional

web

Simple

The website should be simple, easy to get around and intuitive. No unnecessary features, no extra bells and whistles, especially on the word processor. At the same time, we don't want it so simple that we can't do our work.

perm_media

Flexible

Organization is key. Being able to create a document folder system that maintains the structure of the novel while not restricting the ability to rearrange, add or delete documents on the fly.

settings

Functional

Providing tools to make the work eaiser is paramount. Tools like the ability to import, edit and export files, to be able to save my files on my local device, and to be able to work off-line if there is no internet connection available. And, just for kicks, we'll post events like writing workshops, events, and appearances.
